## Clone and review it

Run the following commands to clone and run it.

### To clone the repository

  `git clone git@github.com:HAMMAS-SALEEM/E-Store.git`

### To enter Directory

`cd E-Store`

### To Install Packages

`npm install`

### To run project on local server

`npm start`

### To run project tests

`npm test`

### To open project with vs code 

`code .`
